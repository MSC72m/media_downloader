# Task Specs

## Purpose
This folder stores task-level specs used by agents for non-trivial work.

## Default (Local, Not Versioned)
- Put active specs in `docs/specs/local/`.
- Files in `docs/specs/local/` are gitignored by default.

## Versioned Specs (Optional)
If a spec should be kept in git history, copy/move it to:
- `docs/specs/committed/`

## Template
Start from:
- `docs/specs/TEMPLATE.md`
